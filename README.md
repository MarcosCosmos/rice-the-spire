# Rice The Spire
A [Zebar](https://github.com/glzr-io/zebar) bar themed based on [Slay the Spire 2](https://store.steampowered.com/app/2868840/Slay_the_Spire_2/) using assets sourced via [Spire Codex](https://spire-codex.com/).

It is both a toy project for personal use and a portfolio piece/excuse to explore technologies and, in particular, web accessibilty.

Please note that I am the only intended consumer at this time. Similarly, I would not recommend it as an accessible application as such. I have deliberately made design decisions that create challenges in order to explore them from a programmatic and UX perspective. As such, while feedback would certainly be appreciated, accessibility and other external requirements are non-goals. I believe would be better achieved from the ground up in a distinct project (which I would probably be happy participate in).
